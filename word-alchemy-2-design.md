# 词语炼金术2.0 - Roguelike版本设计案

> 基于全家喜爱的合成玩法，融入Roguelike元素

---

## 🎮 核心概念

**"每一局都是独特的合成冒险"**

- 每局随机生成不同的"目标终极词汇"
- 通过合成解锁新元素，逐步向终极目标迈进
- 失败后可以保留部分"知识"（解锁的配方）

---

## 🏗️ 游戏结构

### 游戏流程
```
选择职业(炼金术士类型) 
  → 获得初始元素包
  → 随机生成"终极目标词汇"
  → 合成解锁路径
  → 到达终极目标获得胜利
  → 解锁新职业/元素包/特殊规则
```

### Roguelike元素

| 元素 | 说明 |
|------|------|
| **随机目标** | 每局随机从词库中选取终极目标（如"永恒"、"宇宙"、"智慧"） |
| **随机初始包** | 5种初始元素包随机发放（自然包、科技包、情感包等） |
| **合成树随机** | 通往终极目标的路径每局不同 |
| **失败惩罚** | 失败后只保留部分已解锁配方 |
| **解锁进度** | 跨局解锁新职业、新元素包、特殊规则 |

## 🎯 职业系统（炼金术士类型）

### 1. 自然炼金师
- **初始元素**: 火、水、土、风
- **特长**: 自然类合成效率+50%
- **特殊能力**: "灵感" - 提示一个可能的合成方向

### 2. 科技工程师
- **初始元素**: 电、金属、代码、能量
- **特长**: 科技类合成解锁更多分支
- **特殊能力**: "分析" - 显示一个元素的合成层级

### 3. 情感导师
- **初始元素**: 爱、梦、记忆、时间
- **特长**: 抽象概念合成更容易
- **特殊能力**: "共鸣" - 随机解锁一个情感类配方

### 4. 混沌术士
- **初始元素**: 随机5个
- **特长**: 合成结果不可预测（有概率获得稀有元素）
- **特殊能力**: "混沌" - 随机转化一个元素为其他元素

## 🌳 合成树设计示例

### 终极目标：永恒
```
永恒
  ├── 时间 + 无限
  │       ├── 沙漏 + 循环
  │       └── 时钟 + 永恒...
  └── 生命 + 死亡
          ├── 人 + 灵魂
          └── 存在 + 终结...
```

### 终极目标：宇宙
```
宇宙
  ├── 空间 + 时间
  │       ├── 虚空 + 维度
  │       └── 星辰 + 历史...
  └── 大爆炸 + 秩序
          ├── 混沌 + 规则
          └── 原子 + 引力...
```

## 🎲 局内随机事件

每合成一定次数触发随机事件：

| 事件 | 效果 |
|------|------|
| **灵感闪现** | 直接显示一个未发现的配方 |
| **元素风暴** | 随机获得一个新元素 |
| **记忆衰退** | 忘记一个已解锁的配方 |
| **双倍炼成** | 接下来3次合成获得双份结果 |
| **神秘商人** | 用分数交换特定元素 |

## 📱 移动端适配

- 一键提示（消耗分数获得合成方向）
- 配方书记录（已解锁的配方可快速重用）
