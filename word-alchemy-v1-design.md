# 词语炼金术 (Word Alchemy) - 游戏设计案

> 游戏地址: https://hiyascott.github.io/game-design-portfolio/games/word-alchemy/
> 类型: 合成解谜
> 核心机制: 通过组合基础元素创造新概念

---

## 🎮 游戏概述

**玩法**: 玩家从4个基础元素开始（火、水、土、风），通过两两合成逐步解锁新元素，最终合成出关卡目标词汇。

**胜利条件**: 成功合成当前关卡的目标词汇

**关卡设计**: 共9关，每关有不同的目标词汇

---

## 🧪 完整合成公式表

- 🔥 火
- 💧 水
- 🌍 土
- 💨 风
- ❤️ 爱

## 🌳 "幸福"的完整合成路径

```
幸福
└── 家庭 + 爱
    └── 房子 + 生命
        ├── 人 + 工具
        │   ├── 生命 + 土
        │   │   └── 植物 + 云
        │   │       ├── 雨 + 土
        │   │       │   └── 云 + 水
        │   │       │       └── 水 + 风
        │   │       └── 植物（来自雨+土）
        │   └── 金属 + 火
        │       └── 石头 + 火
        │           └── 岩浆 + 水
        │               └── 火 + 土
        └── 生命（同上）
```

**简化路径**（最少步骤）：
1. 火 + 水 = 蒸汽
2. 水 + 风 = 云
3. 云 + 水 = 雨
4. 雨 + 土 = 植物
5. 植物 + 云 = 生命
6. 火 + 土 = 岩浆
7. 岩浆 + 水 = 石头
8. 石头 + 火 = 金属
9. 金属 + 火 = 工具
10. 生命 + 土 = 人
11. 人 + 工具 = 房子
12. 房子 + 生命 = 家庭
13. 家庭 + 爱 = 幸福
