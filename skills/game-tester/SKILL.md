# 游戏测试 (Game Tester)

## 概述

游戏手动测试技能，提供标准化的游戏测试流程、测试清单和记录模板。适用于网页游戏的快速功能验证和兼容性测试。

## 测试范围

### 核心测试项

| 测试项 | 说明 |
|-------|------|
| **加载测试** | 游戏能否正常加载，无报错 |
| **第一关验证** | 第一关能否正常开始和完成 |
| **控制响应** | 键盘/鼠标/触摸是否响应 |
| **游戏逻辑** | 核心玩法逻辑是否正常 |
| **音频测试** | 音效和背景音乐是否正常播放 |
| **视觉检查** | UI显示、动画、特效是否正常 |

### 测试清单

- [ ] 游戏能正常加载
- [ ] 第一关正常开始
- [ ] 控制响应及时
- [ ] 游戏逻辑正常
- [ ] 能完成第一关
- [ ] 音频正常播放
- [ ] 无明显视觉问题
- [ ] 跨浏览器兼容

## 使用方法

### 快速测试单个游戏

```bash
# 查看测试清单
./skills/game-tester/test-games.sh
```
