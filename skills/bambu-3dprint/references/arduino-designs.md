## 案例1: Arduino UNO 基础外壳

### 设计要点
```
┌─────────────────────────────────┐
│  ┌───┐                   ┌───┐  │
│  │ ○ │                   │ ○ │  │  ← M3螺丝柱
│  └───┘                   └───┘  │
│                                 │
│         [USB口开口]             │
│                                 │
│  ┌───┐                   ┌───┐  │
│  │ ○ │                   │ ○ │  │
│  └───┘                   └───┘  │
└─────────────────────────────────┘
```

### Fusion 360 设计步骤
1. **草图**: 绘制68.6×53.4矩形
2. **偏移**: 内缩1mm作为内部空间
3. **拉伸**: 外框2mm厚，侧壁10mm高
4. **螺丝柱**: 在四角放置Ø6mm圆柱，中心Ø2.8mm孔
5. **开口**: USB口、电源口位置开槽

### 打印设置
- 层高: 0.20mm
- 填充: 20%
- 支撑: 无需（悬空<45°）
- 壁厚: 3层

## 案例2: 传感器支架

### 超声波传感器(HC-SR04)支架

```
    传感器尺寸: 45mm × 20mm
    
    支架设计:
    ┌─────────────────┐
    │  ╔═══════════╗  │  ← 传感器卡槽
    │  ║ ○     ○   ║  │    深度: 5mm
    │  ╚═══════════╝  │
    │       │         │
    │    [支柱]       │  ← 高度可调
    │       │         │
    │    ┌───┐        │  ← 底座带螺丝孔
    └────┘   └────────┘
```

### 关键尺寸
- 卡槽内宽: 45.5mm（预留0.5mm间隙）
- 卡槽深度: 5mm
- 支柱高度: 根据安装位置调整

## 案例3: 步进电机支架

### NEMA 17 标准尺寸
- 正面: 42.3mm × 42.3mm
- 安装孔: 31mm × 31mm（M3）
- 轴径: 5mm

### 支架设计
```
    侧视图:
    ┌────────────┐
    │   ┌──┐     │  ← 电机安装面
    │   │██│     │    厚度: 5mm
    │   └──┘     │
    │    │       │
    │   [支架]   │  ← 加强筋
    │    │       │
    └───┬┴┬──────┘  ← 底座固定
```

### 强度优化
- 使用三角形加强筋
- 填充密度: 50%
- 壁厚: 4层
- 材料: PETG（比PLA强度高）

## 案例4: 模块化外壳系统

### 设计理念
使用统一接口，多个模块可拼接。

```
    标准模块: 50mm × 50mm × 25mm
    
    连接方式:
    滑轨式:          卡扣式:
    ┌────┐           ┌──┐
    │ ═══╪════┐      │ ▓│ ← 公扣
    └────┘    │      └──┘
              │        │
         ┌────┤      ┌──┐
         │════╪═══│  │ ░│ ← 母扣
         └────┘      └──┘
```

### 接口规格
- 滑轨: T型槽，宽8mm，深5mm
- 卡扣: 凸起3mm × 5mm，间隙0.2mm

## 案例5: 线缆管理

### 线夹设计
```
    侧视图:           俯视图:
    ┌──┐              ┌──────┐
    │  │ ← 开口       │ ◯◯◯◯ │  ← 4线孔
    │  │   弹性设计    │      │
    └──┘              └──┬───┘
       │                 │
      [底座]          [安装孔]
```

### 参数
- 线孔直径: 根据线缆调整（常用5mm、8mm）
- 开口宽度: 线径的80%（卡紧但不伤线）
- 材料: TPU柔性打印

## 设计检查清单

在导出打印前确认：

- [ ] **螺丝孔径**: 标准M3用2.8mm（PLA收缩余量）
- [ ] **公差预留**: 活动件间隙0.2-0.3mm
- [ ] **壁厚**: 最少1.2mm（3层@0.4mm喷嘴）
- [ ] **支撑检查**: 悬空面>45°需支撑
- [ ] **打印方向**: 强度方向与受力方向一致
- [ ] **热 bed 附着**: 大面积底座考虑裙边/ Raft
