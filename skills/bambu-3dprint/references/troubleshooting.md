# 故障排除完整指南

## 第一层问题

### 不粘床

**现象**: 打印刚开始就脱落或移位

**原因与解决**:

| 原因 | 解决方法 |
|------|----------|
| 热床未调平 | 运行自动调平程序 |
| 打印面板脏 | 用酒精擦拭，或水洗（PEI板） |
| 首层间隙大 | 调整Z-offset（减小0.05-0.1mm） |
| 温度过低 | PLA热床60°C，PETG 70-80°C |
| 耗材问题 | 换品牌或干燥耗材 |

**拓竹特别建议**:
- 使用官方贴纸或纹理PEI板
- 首层速度降至50mm/s以下
- 开启"热床调平"选项

---

## 层间问题

### 层分离 / 开裂

**现象**: 模型像千层饼一样分层

**解决**:
- 提高热床温度（+10°C）
- 提高喷头温度（+5-10°C）
- 降低冷却风扇速度（50-70%）
- 检查层高是否过大（建议≤0.28mm）

### 层错位

**现象**: 某层突然偏移，模型"错层"

**原因**:
- 皮带松动
- 打印速度过快
- 喷头碰撞模型

**解决**:
1. 检查X/Y轴皮带张力
2. 降低打印速度
3. 检查模型是否有悬空导致喷头碰撞

---

## 表面质量问题

### 拉丝 /  Stringing

**现象**: 移动时拉出细丝，影响表面

**调整参数**:
```
Retraction Distance: 0.8mm → 1.2mm
Retraction Speed: 30mm/s → 50mm/s
Travel Speed: 200mm/s → 300mm/s
Temperature: -5°C (降低温度减少流动)
```

### 枕形效应 (Pillow Effect)

**现象**: 顶层凹陷，像枕头

**解决**:
- 增加顶层数量（3层 → 5层）
- 增加填充密度（15% → 25%）
- 降低喷头温度

### 层纹明显

**现象**: 侧面可见明显水平线

**原因**:
- 层高设置大
- 机械振动

**解决**:
- 降低层高（0.2mm → 0.12mm）
- 检查螺丝是否紧固
- 使用外墙慢速（30mm/s）

---

## 支撑相关问题

### 支撑粘太牢

**解决**:
- 增加支撑顶部Z距离（0.2mm → 0.3mm）
- 减少支撑密度（50% → 30%）
- 选择"树状支撑"（接触面积小）

### 支撑脱落

**解决**:
- 增加支撑底座（Raft）
- 增加支撑与模型间距（不可过大）
- 检查首层附着

---

## 耗材问题

### 堵头

**现象**: 不出料或出料断断续续

**解决步骤**:
1. 升温至220°C，手动挤出测试
2. 如果失败，拆下喷头清理
3. 检查耗材是否打结
4. 检查进料齿轮是否卡料屑

### 耗材受潮

**现象**: 打印时噼啪声、气泡、表面粗糙

**解决**:
- 使用耗材干燥箱
- 烤箱烘干（50-60°C，4-6小时）
- 使用新开封耗材测试

### 耗材缠绕

**现象**: 进料突然受阻

**解决**:
- 整理料盘，确保顺畅
- 使用料盘支架减少阻力
- 检查料盘是否交叉缠绕

---

## 拓竹特有功能问题

### AMS换料失败

**现象**: 多色打印时换料卡住

**解决**:
1. 检查料管是否弯折
2. 清洁AMS进料轮
3. 确保耗材末端剪平
4. 避免使用易碎的支撑材料

### AI检测误报

**现象**: 正常打印被判定为失败暂停

**解决**:
- 调整AI灵敏度
- 某些模型特征（如密集支撑）可能触发误报
- 复杂模型可关闭AI检测

---

## 紧急停止处理

如果打印严重出错：

1. **立即停止**: 按打印机屏幕暂停/停止
2. **不要直接拔电源**: 等喷头冷却
3. **清理残余**: 热床冷却后清理残留料
4. **分析原因**: 对照本指南找出问题
5. **调整后重试**: 修改参数再打印
