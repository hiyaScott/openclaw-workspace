# Bambu 3D打印技能

> 与拓竹3D打印机一起创造实体世界。

## 快速上手

### 软件安装

1. **下载 Bambu Studio** (官方切片软件)
   - 官网：https://bambulab.cn/zh-cn/download
   - 支持：Windows / macOS

2. **初次设置**
   - 地区选择："中国内地大陆"（可使用手机号登录）
   - 安装时勾选所有打印机型号网络插件
   - 绑定打印机：通过Bambu Handy App扫描二维码

### 打印流程（5步）

```
获取模型 → 导入切片 → 设置参数 → 切片预览 → 发送打印
```

1. **获取模型**：从内置模型库或第三方网站下载 (.stl/.3mf/.obj)
2. **导入切片**：拖入Bambu Studio或点击"添加"
3. **设置参数**：选择打印机型号、耗材类型(PLA/PETG/TPU等)、层高
4. **切片预览**：检查支撑、打印时间、耗材用量
5. **发送打印**：WiFi直传或导出到SD卡

## Arduino结构件设计

### 常用连接方式

```
[卡扣式]  适合快速拆装
    ┌───┐
    │ ▓ │ ← 凸起卡扣
    └───┘
      ↓
    ┌───┐
    │ ░ │ ← 凹槽
    └───┘

[螺丝柱]  适合固定PCB
    ╔═══╗
    ║ ○ ║ ← M3螺丝孔
    ╚═══╝
      │
    底座

[滑轨式]  适合模块化
    ┌─────────┐
    │  ═════  │ ← 滑槽
    └─────────┘
         ↑
    ┌────┐
    │════┤ ← 滑块
    └────┘
```

### 设计参数速查

| 项目 | 推荐值 | 说明 |
|------|--------|------|
| 螺丝孔径 | 2.8mm (M3) | PLA预留0.2mm收缩余量 |
| 卡扣间隙 | 0.2-0.3mm | 太紧难装，太松松脱 |
| 壁厚 | ≥1.2mm | 2-3层墙，保证强度 |
| 底座厚度 | ≥2mm | 防止翘边 |
| 支撑角度 | >45°需支撑 | 悬空面需要支撑 |

### Arduino外壳设计模板

```openscad
// Arduino UNO 外壳参数
board_width = 53.4;   // 主板宽度
board_length = 68.6;  // 主板长度
wall_thickness = 2;   // 壁厚
height = 15;          // 外壳高度

// 螺丝柱位置（UNO标准孔位）
holes = [
    [2.5, 2.5],
    [2.5, 66.1],
    [50.9, 2.5],
    [50.9, 66.1]
];
```

## 切片参数指南

### 耗材选择

| 材料 | 特点 | 用途 | 注意 |
|------|------|------|------|
| **PLA** | 易打印、无毒、颜色丰富 | 原型、装饰、外壳 | 不耐热(60°C以下) |
| **PETG** | 强度高、耐冲击、透明可选 | 结构件、功能性零件 | 易拉丝，需调 retractions |
| **TPU** | 柔性、弹性 | 垫圈、减震、软外壳 | 打印慢，易堵头 |
| **ABS/ASA** | 耐热、强度高 | 户外、高温环境 | 需封箱，气味大 |

### 关键参数

```
[质量 vs 速度]
层高: 0.08mm (精细) ←→ 0.28mm (快速)
速度: 50mm/s (稳定) ←→ 500mm/s (拓竹极速)
填充: 15% (轻量) ←→ 50% (强度)
```

| 参数 | 默认值 | 调整建议 |
|------|--------|----------|
| 层高 | 0.20mm | 精细0.12mm，快速0.28mm |
| 墙厚 | 3层 | 受力件4-5层 |
| 填充密度 | 15% | 结构件30-50% |
| 填充图案 | Grid | 强度用Honeycomb，快速用Lines |
| 支撑 | 树状 | 自动检测，复杂模型用手动 |

## 常见问题排查

### 打印失败检查清单

- [ ] **第一层不粘** → 调平热床、清理面板、涂胶/喷雾
- [ ] **层间分离** → 提高热床温度、检查层高
- [ ] **拉丝严重** → 调整Retraction距离/速度
- [ ] **支撑难拆** → 增加支撑间隙(Z距离)
- [ ] **表面粗糙** → 降低打印速度、检查耗材湿度

### 拓竹特有功能

| 功能 | 说明 |
|------|------|
| **AMS** | 自动换料系统，支持多色/多材料打印 |
| **AI检测** | 自动检测 spaghetti 和打印失败 |
| **延时摄影** | 自动生成打印过程延时视频 |
| **远程监控** | 通过手机App随时查看打印状态 |
