# 剧本案例分析

## 案例1: 三幕结构分析 - 《星球大战：新希望》

### 第一幕：建置 (p1-p30)
- **开场**: 帝国星际驱逐舰追击反抗军飞船
- **人物介绍**: 卢克、欧比旺、莱娅公主、汉·索罗
- **激励事件**: 卢克发现R2-D2中的全息信息
- **转折点1**: 卢克发现叔叔婶婶被杀，决定跟欧比旺去救公主

### 第二幕：对抗 (p30-p90)
- **游戏时间**: 塔图因出发、汉·索罗加入、死星救援
- **中点**: 救出莱娅公主，但欧比旺与达斯·维达对决牺牲
- **一无所有**: 反抗军基地被发现，死星逼近
- **灵魂黑夜**: 卢克失去导师，感到迷茫

### 第三幕：解决 (p90-p120)
- **转折点2**: 卢克决定加入反抗军攻击死星
- **高潮**: 死星壕沟追逐战，欧比旺的声音指引卢克使用原力
- **结局**: 死星被摧毁，卢克获颁奖章

## 案例2: 分支叙事分析 - 《底特律：变人》

### 叙事结构特点
- 三条主角线并行
- 关键选择影响整体走向
- 角色可能死亡，故事继续

### 设计亮点
1. **状态跟踪系统**: 角色关系、公众舆论、个人特质
2. **蝴蝶效应**: 小选择在后期产生大影响
3. **道德困境**: 没有"正确"答案的选择

### 设计教训
- 分支过多导致部分线路叙事薄弱
- 需要平衡分支数量与叙事深度
- 确保每个主要分支都有足够的场景支撑

## 案例3: 环境叙事分析 - 《艾迪芬奇的记忆》

### 叙事手法
- 每个房间讲述一个家庭成员的故事
- 无直接对话，通过物品、装饰、环境互动叙事
- 独特的"玩法"呈现不同人物的死亡

### 关卡设计要点
- **Barbara的房间**: 漫画风格呈现，与角色职业呼应
- **Lewis的房间**: 罐头工厂+幻想世界，双重视觉呈现
- **Milton的房间**: 未完成，暗示失踪，留白叙事

### 可借鉴技巧
1. 房间装饰反映人物性格
2. 视觉风格随叙事变化
3. 可交互3D文字引导路径
4. 象征物贯穿始终

## 案例4: 魂类箱庭分析 - 《黑暗之魂：城外不死镇》

### 空间结构
```
         飞龙桥
            ↑
    牛头恶魔塔楼
            ↑
   不死镇下层(环)
            ↑
    篝火(桥底)
```

### 设计要点
1. **环形路径**: 从左侧出发，绕一圈从右侧塔楼上到飞龙桥
2. **捷径**: 踢落梯子从飞龙桥直达桥底篝火
3. **密度变化**: 城镇复杂探索 → 塔楼线性上升 → Boss战
4. **视觉锚点**: 飞龙桥、塔楼作为方向参考

### 叙事元素
- 环境暗示曾经繁荣的不死镇
- 散落的信息碎片讲述不死诅咒
- NPC分布创造"还有人活着"的错觉
