# 战棋数值平衡 (Numerical Balance)

## 战棋数值特点

### 与其他游戏类型的差异

```
对比:

MMORPG:
├─ 连续战斗，数值平滑
├─ 强调DPS/HPS
└─ 成长感重要

卡牌游戏:
├─ 离散战斗，随机抽牌
├─ 强调牌效和资源管理
└─ 构筑感重要

战棋SRPG:
├─ 离散战斗，每关独立
├─ 强调位置/策略/单位搭配
├─ 单个单位价值高 (permadeath)
└─ 策略验证 > 数值验证
```

### 核心平衡原则

```
策略优先:
- 策略正确可以越级挑战
- 策略错误会被同级击败
- 数值提供容错空间而非决定性优势

单位价值:
- 每个单位都珍贵 (permadeath设计)
- 损失评估是策略一部分
-  recovery成本高昂
```

## 基础数值公式

### 伤害公式设计

#### 减法公式 (日式SRPG主流)

```
伤害 = max(1, 攻击力 - 防御力) × 克制系数 × 其他系数

示例:
攻击力: 50
防御力: 30
克制系数: 1.3 (兵种克制)
暴击: 1.5

伤害 = (50 - 30) × 1.3 × 1.5 = 39

优点:
├─ 直观易懂
├─ 防御价值明确
└─ 低攻打高防有意义

缺点:
├─ 攻防差过大时伤害固化
├─ 需要精心平衡数值范围
└─ 容易出现不破防情况
```

#### 除法公式 (更平滑)

```
伤害 = 攻击力 × (攻击力 / (攻击力 + 防御力 × K)) × 其他系数

或简化:
伤害 = 攻击力 × (1 - 防御力/(防御力+100))

示例:
攻击力: 50
防御力: 30

伤害 = 50 × (50 / (50 + 30)) = 31.25

优点:
├─ 平滑递减
├─ 不会出现不破防
└─ 攻防价值相对平衡

缺点:
├─ 不够直观
├─ 防御收益递减
└─ 需要调试K值
```

### 命中率公式

```
命中率 = min(95%, max(5%, 技巧 - 速度 + 武器命中 + 地形 + 其他))

要素:
├─ 技巧: 攻击方属性
├─ 速度: 防御方属性
├─ 武器命中: 武器基础命中
├─ 地形: 地形闪避修正
└─ 其他: 技能/状态修正

设计考虑:
- 上限95%: 保留不确定性
- 下限5%: 保留希望
- 速度影响闪避: 鼓励速度投资
```

### 暴击公式

```
暴击率 = max(0, (技巧 - 幸运)/2 + 武器暴击)

暴击伤害:
├─ 通常: 2倍伤害
├─ 部分游戏: 3倍 (火纹)
└─ 可配合: 暴击抗性/免疫

设计考虑:
- 暴击是惊喜而非期望
- 基础暴击率3-10%
- 高暴击职业特色鲜明
```

## 属性成长设计

### 成长率系统 (火纹模式)

```
每个角色/职业有成长率:
├─ 每次升级随机提升属性
├─ 成长率是概率 (如HP 80% = 80%概率+1)
├─ 个体差异带来角色特色
└─ 凹点文化来源

平衡控制:
├─ 总成长率控制 (早熟/晚成)
├─ 成长率分配 (专攻某项)
└─ 上限控制 (避免无限成长)
```

### 固定成长 (手游常见)

```
每级固定提升:
├─  predictable成长
├─ 易于规划
├─ 减少随机性
└─ 适合长线养成

设计要点:
- 不同职业成长曲线不同
- 提供重置/优化机会
- 配合技能解锁节奏
```

## 战斗平衡

### TTK (Time To Kill) 控制

```
击杀期望回合数:

脆皮单位:
├─ 被同级攻击: 2-3次
├─ 被克制攻击: 1-2次
└─ 策略意义: 需要保护

坦克单位:
├─ 被同级攻击: 4-6次
├─ 被克制攻击: 2-3次
└─ 策略意义: 前线抗压

设计目的:
- 战斗不会太快结束
- 每个单位都有存在感
- 失误有挽回空间
```

### 职业平衡

```
三角平衡:

剑士 vs 枪兵 vs 骑兵

克制效果:
├─ 克制方: +20-30% 伤害
├─ 被克方: -20-30% 伤害
└─ 平衡: 任何职业都有优劣势

非克制战斗:
├─ 职业特色发挥作用
├─ 技能/装备影响更大
└─ 策略选择更重要
```

### 射程与移动力平衡

```
移动与射程关系:

近战单位 (1格射程):
├─ 高移动力 (5-6格)
├─ 高防御/生命
├─ 高伤害
└─ 需要接近敌人

远程单位 (2-3格射程):
├─ 中等移动力 (3-4格)
├─ 低防御/生命
├─ 中等伤害
└─ 需要保持距离

超远程 (4格+射程):
├─ 低移动力 (2-3格)
├─ 极低防御
├─ 需要保护
└─ 射程是生存手段

设计原则:
- 移动力与射程成反比
- 射程优势需要代价
- 地形影响射程价值
```

## 养成与关卡匹配

### 难度曲线设计

```
推荐战力系统:

关卡难度 = 推荐等级 × 基准属性

实际体验:
├─ 战力达标: 适当策略可通过
├─ 战力超标20%: 数值碾压
├─ 战力不足20%: 需要精密策略
└─ 战力不足50%: 极难/不可能

设计目的:
- 给玩家成长感
- 但策略仍能弥补数值差距
- 不同付费层级都有挑战
```

### 属性验证节奏

```
成长验证:

早期 (1-10级):
├─ 属性差异小
├─ 策略影响大
└─ 教学关卡为主

中期 (10-30级):
├─ 属性差异显现
├─ 职业特色形成
└─ 策略配合数值

后期 (30级+):
├─ 属性差异大
├─ 养成深度体现
└─ 高难度策略挑战
```

## 手游SRPG数值规划

### 长线养成数值

```
养成线设计:

角色等级:
├─ 满级: 60-80级
├─ 前期快 (1-40)
├─ 后期慢 (40-60)
└─ 极慢 (60+)

装备强化:
├─ 等级: +1 到 +15/+20
├─ 前期成功率100%
├─ 后期成功率递减
└─ 失败惩罚 (降级/破碎)

技能升级:
├─ 等级: 1-5级或1-10级
├─ 材料需求递增
└─ 效果提升非线性
```

### 付费与免费平衡

```
战力差距控制:

同等级对比:
├─ 免费玩家: 基准战力100%
├─ 月卡玩家: 110-120%
├─ 中氪玩家: 130-150%
└─ 重氪玩家: 180-200%

设计原则:
- 免费玩家能通关主线
- 付费玩家更轻松/更快
- 策略可以弥补付费差距
- 高难内容需要策略+数值

时间换算:
├─ 1元 ≈ 10-15分钟游戏时间
├─ 月卡性价比最高
└─ 重氪边际效应递减
```

## 数值调试方法

### 战斗模拟

```
Excel/Google Sheets:

建立模型:
├─ 输入: 攻防属性
├─ 计算: 伤害/命中/击杀回合
└─ 输出: TTK/胜率

批量测试:
├─ 不同职业组合
├─ 不同属性配比
└─ 找出最优/最差情况
```

### 玩家测试

```
观察指标:
├─ 通关时间
├─ 单位损失数量
├─ 重来次数
├─ 使用的策略多样性
└─ 挫败/成就感反馈

调优方向:
├─ 太难: 降低数值/增加提示
├─ 太易: 增加机制/提高AI
└─ 单调: 增加变数/机制
```
