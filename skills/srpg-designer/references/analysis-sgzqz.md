# 《三国志·战棋版》战斗系统深度分析

## 分析框架

本分析综合运用以下能力维度：
- **网络游戏设计师**: 系统架构、数值平衡、长线运营设计
- **游戏设计**: 玩法循环、心流体验、决策设计
- **战棋游戏设计师**: 网格战斗、职业体系、策略维度

---

## 一、核心设计哲学

### 1.1 SLG+SRPG融合

```
三国志战棋版 = 三国志战略版(沙盘SLG) + 战棋战斗(SRPG)
```

**双重体验**:
- 大地图: 资源争夺、联盟战争、策略博弈
- 战斗内: 回合制战棋、兵种克制、技能配合

### 1.2 核心循环

```
养成 → 配将 → 战法搭配 → 战棋战斗 → 获得资源 → 进一步强化
```

## 二、兵种系统 —— 四兵种循环克制

### 2.1 基础兵种

| 兵种 | 特性 | 移动 | 被克 |
|------|------|------|------|
| **骑兵** | 高机动、冲锋 | 5格 | 枪兵 |
| **盾兵** | 高防御、坚守 | 3格 | 骑兵 |
| **枪兵** | 中距离、反骑 | 3格 | 弓兵 |
| **弓兵** | 远程、高输出 | 3格 | 盾兵 |

### 2.2 克制关系

```
骑兵 → 盾兵 → 弓兵 → 枪兵 → 骑兵
```

**克制效果**: 造成伤害提升，受到伤害降低

### 2.3 高级兵种

| 高级兵种 | 特性 | 解锁条件 |
|---------|------|---------|
| **西凉铁骑** | 剩余移动力免伤+受伤降低 | 马腾/马超/吕布等+5骑兵良将 |
| **虎豹骑** | 前两回合伤害提升 | 曹纯/曹操/关羽等+5骑兵良将 |
| **精锐骑兵** | 攻防属性提升 | 10骑兵良将 |

## 三、战法系统 —— 技能核心

### 3.1 战法结构

每个武将可携带:
- **1个自带战法** (武将专属)
- **2个学习战法** (传承/事件战法)

### 3.2 战法类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **被动** | 持续生效 | 【百步绝缨】增加射程和伤害 |
| **主动** | 需主动释放 | 【八门金锁】辅助技能 |
| **突击** | 普攻后概率触发 | 【困龙缚虎】额外伤害 |
| **指挥** | 战斗开始生效 | 【御敌屏障】减伤 |
| **阵法** | 影响全队 | 【行守相扶】分摊伤害 |
| **兵书** | 被动加成 | 【通粮致谷】回血 |

### 3.3 战法传承系统

- 消耗武将传承战法
- 传承后武将消失
- 可"再起"恢复武将（有次数限制）

## 四、核心战斗机制

### 4.1 回合制框架

```
我方回合 → 操作所有单位 → 敌方回合 → 敌方操作 → 循环
```

**行动点**:
- 移动 (根据移动力)
- 行动 (普攻/战法/待机)

### 4.2 属性系统

| 属性 | 作用 |
|------|------|
| **武力** | 物理攻击力 |
| **统帅** | 物理防御力 |
| **智力** | 魔法攻击+防御 |
| **兵力** | 血量（决定伤害） |

**伤害公式**:
```
伤害 ∝ 武将强度 × 当前兵力
```

### 4.3 再动机制

**刘禅【放权】**:
- 目标友军获得【再动】
- 但下一轮进入【休息】状态
- 施法者属性降低15点

**战术价值**:
- 一回合爆发两次
- 配合高爆发武将
- 创造"伪双动"

## 七、设计亮点总结

### 7.1 成功要素

1. **SLG+SRPG融合**: 大地图策略+战棋战斗
2. **战法系统**: 丰富的技能搭配
3. **事件战法**: 历史感+强力奖励
4. **兵种克制**: 简单明了的策略基础
5. **再动机制**: 创造爆发机会

### 7.2 可借鉴设计

- 战法传承系统增加收集要素
- 事件战法增加历史沉浸感
- 再动机制创造战术多样性
- 属性与兵力挂钩强调养成价值
