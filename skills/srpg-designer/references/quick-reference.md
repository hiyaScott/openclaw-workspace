## 一、技能倍率速查表

### 1.1 单体技能倍率

| 类型 | 最低 | 常见 | 最高 | 设计建议 |
|------|------|------|------|----------|
| 普攻 | 1.0x | 1.0x | 1.0x | 基准值 |
| 基础技能 | 1.2x | 1.3-1.5x | 1.6x | 主力输出 |
| 高伤技能 | 1.5x | 1.6-1.8x | 2.0x | 核心爆发 |
| 极限技能 | 1.8x | 2.0-2.5x | 3.0x | 限制条件触发 |

**五游戏对比**:
- 天地劫: 单体1.5-1.8倍，气力技可达2.0倍
- 梦幻模拟战: 单体1.3-1.8倍，克制+20%
- 铃兰之剑: 单体1.2-1.6倍，考虑职业克制
- 三国志战棋版: 单体1.4-1.8倍，战法联动
- 三国望神州: 单体1.5-1.8倍，合击加成

### 1.2 AOE技能倍率

| 范围 | 最低 | 常见 | 最高 | 设计建议 |
|------|------|------|------|----------|
| 小范围(3格) | 0.3x | 0.5-0.6x | 0.8x | 控制为主 |
| 中范围(5格) | 0.4x | 0.6-0.7x | 0.9x | 平衡型 |
| 大范围(7格+) | 0.3x | 0.5x | 0.7x | 纯伤害较低 |

**五游戏对比**:
- 天地劫: AOE 0.5-0.7倍，地形效果加成
- 梦幻模拟战: AOE 0.4-0.7倍，黑洞/天罚标准
- 铃兰之剑: AOE 0.5-0.7倍，地形互动
- 三国志战棋版: AOE 0.5-0.8倍，火烧连营
- 三国望神州: AOE 0.6-0.8倍，奇谋加成

### 1.3 追击/连击倍率

| 类型 | 倍率 | 触发条件 | 代表角色 |
|------|------|----------|----------|
| 追击 | 0.3-0.5x | 攻击后触发 | 韩千秀 |
| 连击 | 0.5-0.7x | 技能附带 | 太史慈 |
| 再动 | 1.0x | 击杀/条件触发 | 冰璃、雪莉 |
| 合击 | 1.0-1.2x | 相邻友方 | 吕布 |

## 二、移动/射程边界值

### 2.1 移动力基准

| 类型 | 移动力 | 特性 | 代表 |
|------|--------|------|------|
| 重甲/盾兵 | 3-4格 | 高防御 | 泰登、曹仁 |
| 步兵/侠客 | 4-5格 | 平衡 | 艾尔文、殷剑平 |
| 骑兵/御风 | 6-8格 | 高机动 | 利昂、曹沁 |
| 飞行单位 | 5-6格 | 无视地形 | 雪莉、露娜 |
| 突进技能 | +2-4格 | 一次性 | 突击、突进 |

**设计上限**: 单次移动不超过8格，突进技能不超过5格

### 2.2 射程边界值

| 类型 | 射程 | 特性 | 代表 |
|------|------|------|------|
| 近战 | 1格 | 基础 | 所有近战 |
| 长柄 | 1-2格 | 中距离 | 枪兵 |
| 弓箭 | 2-3格 | 远程 | 黄忠、娜姆 |
| 法术 | 1-3格 | 通常有最小射程 | 法师 |
| 狙击 | 3-5格 | 超远程 | 黄忠(定军山) |

**设计上限**: 常规射程不超过3格，狙击技能可达5格

### 2.3 再动机制

| 类型 | 触发条件 | 限制 | 代表角色 |
|------|----------|------|----------|
| 击杀再动 | 击杀敌人 | 每回合1-2次 | 雪莉、科尔 |
| 技能再动 | 使用技能后 | 概率/条件 | 艾尔文、冰璃 |
| 被动再动 | 特定条件 | 冷却/限制 | 泽瑞达、胧 |
| 合击再动 | 相邻攻击 | 需友方配合 | 吕布 |

## 四、常见设计模式模板

### 4.1 技能设计模板

#### 模板A: 标准输出技能
```
技能名称: [名称]
类型: 单体/范围
倍率: 1.5倍 (单体) / 0.6倍 (范围)
效果: [基础效果]
冷却: 2-3回合
消耗: [如果有]

设计要点:
- 倍率符合类型标准
- 冷却与强度匹配
- 有明确的定位
```

#### 模板B: 控制技能
```
技能名称: [名称]
类型: 单体/范围控制
倍率: 0.8-1.2倍 (低伤害)
效果: [控制效果] + [debuff]
持续时间: 1-2回合
命中率: 70-90%

设计要点:
- 控制效果优先
- 伤害为辅助
- 命中率需平衡
```

#### 模板C: 被动技能
```
技能名称: [名称]
类型: 被动/天赋
触发条件: [条件]
效果: [效果描述]
冷却: [如果有]

设计要点:
- 触发条件明确
- 效果与定位匹配
- 避免过强/过弱
```

### 4.2 角色设计模板

#### 模板A: 输出核心
```
定位: 主C/副C
属性: 攻击型
核心机制: [再动/暴击/连击等]

技能配置:
1. 主力输出技能 (1.6-1.8倍)
2. 范围/AOE技能 (0.6-0.7倍)
3. 被动增益/生存技能
4. 大招/气力技 (1.8-2.0倍)

设计要点:
- 有明确的输出手段
- 有生存保障
- 与其他角色有配合
```

#### 模板B: 坦克/护卫
```
定位: 主T/功能T
属性: 防御型
核心机制: [护卫/减伤/反震等]

技能配置:
1. 护卫/援护技能
2. 控制/减益技能
3. 被动生存技能
4. 团队增益/辅助

设计要点:
- 能有效保护队友
- 有生存能力
- 不只是肉盾，要有功能
```

#### 模板C: 治疗/辅助
```
定位: 主奶/功能辅
属性: 辅助型
核心机制: [治疗/再动/增益等]

技能配置:
1. 单体/群体治疗
2. 驱散/净化技能
3. 增益/buff技能
4. 特殊功能技能

设计要点:
- 治疗量充足
- 有功能多样性
- 不是纯奶，要有特色
```

### 4.3 关卡设计模板

#### 模板A: 教学关
```
目标: 教学新机制
敌人: 2-3个弱敌人
机制: 引入1个新机制
难度: 简单

设计要点:
- 清晰的教学引导
- 安全的试错空间
- 明确的胜利条件
```

#### 模板B: 标准战斗关
```
目标: 歼灭/守护/撤离
敌人: 5-8个敌人，1-2个精英
机制: 2-3个机制组合
难度: 中等

设计要点:
- 敌人配置合理
- 地形利用空间大
- 有多种解法
```

#### 模板C: BOSS关
```
目标: 击败BOSS
敌人: 1个BOSS + 若干小怪
机制: BOSS专属机制
难度: 困难

设计要点:
- BOSS机制清晰
- 有明确的应对策略
- 有容错空间
```
