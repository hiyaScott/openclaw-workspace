# 战斗系统设计 (Combat Design)

## 职业/兵种设计

### 基础职业三角

```
经典武器三角 (火纹模式):

        剑/步兵
       /       \
      /         \
   枪/长柄 ───── 斧/重武

克制关系: 剑克斧，斧克枪，枪克剑
克制加成: 通常 1.2x - 1.5x 伤害

变体:
- 魔法三角: 火-雷-风
- 近战-远程-魔法
- 飞行-弓箭-魔法
```

### 职业类型设计

```
标准职业分类:

坦克型 (T):
├─ 高防御/生命
├─ 中等攻击
├─ 低移动力
├─ 技能: 嘲讽/护盾/减伤
└─ 代表: 骑士、重甲兵

输出型 (DPS):
├─ 高攻击
├─ 低防御
├─ 中等移动力
├─ 技能: 高伤害/暴击/连击
└─ 代表: 剑士、刺客、法师

远程型:
├─ 高射程
├─ 低防御
├─ 低移动力
├─ 技能: 范围攻击/精准打击
└─ 代表: 弓箭手、狙击手

辅助型 (Support):
├─ 低攻击/防御
├─ 中等移动力
├─ 技能: 治疗/增益/减益
└─ 代表: 牧师、吟游诗人

特殊型:
├─ 飞行单位 (无视地形)
├─ 盗贼 (高移速/开锁)
├─ 工程 (陷阱/机关)
└─ 召唤 (召唤物作战)
```

### 职业进阶系统

```
转职树设计 (梦幻模拟战模式):

        战士
       /    \
    剑士    枪兵
    /  \    /  \
  剑圣  英雄  骑士  龙骑

设计要点:
- 多分支提供差异化成长
- 进阶解锁强力技能
- 跨系转职保留部分能力
- 终极职业有独特定位
```

## 技能系统设计

### 技能类型

```
主动技能:
├─ 伤害技能: 直接造成伤害
├─ 控制技能: 眩晕/冰冻/沉默
├─ 位移技能: 冲锋/后撤/换位
├─ 增益技能: 加攻/加防/加速
└─ 减益技能: 降攻/降防/中毒

被动技能:
├─ 属性加成: 常驻属性提升
├─ 触发效果: 特定条件触发
├─ 光环效果: 影响周围单位
└─ 特性能力: 无视地形/飞行

连携技能:
├─ 双人连携: 两个单位同时攻击
├─ 羁绊技能: 好感度解锁
└─ 阵法技能: 特定站位触发
```

### 技能设计原则

```
技能CD与消耗:
- 强力技能: 长CD/高消耗/有限使用次数
- 普通技能: 短CD/低消耗
- 终极技能: 一局一次或极长CD

技能范围设计:
├─ 自身: 增益/治疗自己
├─ 单体: 1个目标
├─ 直线: 前方X格
├─ 扇形: 锥形范围
├─ 圆形: 以自己或目标为中心
└─ 全场: 影响所有单位

技能升级:
├─ 数值提升: 伤害/效果加强
├─ 范围扩大: 影响更多目标
├─ 副作用减少: CD减少/消耗降低
└─ 新效果: 增加额外效果
```

## 战斗机制深度

### 位置与朝向

```
面向系统 (皇家骑士团/XCOM):

伤害加成:
├─ 正面攻击: 标准伤害
├─ 侧面攻击: +10% 伤害
└─ 背面攻击: +25% 伤害

防御加成:
├─ 正面受击: 标准防御
├─ 侧面受击: -10% 防御
└─ 背面受击: -25% 防御

战术意义:
- 保护脆弱单位的背后
- 绕后偷袭高价值目标
- 阵型保持重要性
```

### 掩体系统 (XCOM模式)

```
掩体类型:

半掩 (Half Cover):
├─ 提供 +20% 防御
├─ 可被侧翼绕过
└─ 可被摧毁

全掩 (Full Cover):
├─ 提供 +40% 防御
├─ 侧面/背面仍有效
└─ 更难以摧毁

掩体破坏:
├─ 爆炸物可以破坏掩体
├─ 破坏后单位暴露
└─ 环境互动创造新掩体
```

### 高低差系统

```
《皇家骑士团》高低差:

战斗影响:
├─ 高打低: +伤害, +命中
├─ 低打高: -伤害, -命中
├─ 超过最大高低差: 无法攻击
└─ 跌落: 高度差过大时受伤

移动影响:
├─ 向上移动: 额外移动力消耗
├─ 向下移动: 正常或降低消耗
└─ 跳跃能力: 部分职业可跨越

抛射物:
├─ 弓箭受重力影响，高处射程更远
├─ 子弹/魔法直线，高低差影响命中
└─ 投石/抛物线可越过障碍
```

## 兵种携带系统

```
《梦幻模拟战》佣兵系统:

机制:
├─ 英雄作为主将
├─ 可携带一定数量的佣兵
├─ 佣兵有独立属性和技能
├─ 战斗时主将+佣兵作为一个单位行动
└─ 佣兵死亡可重新招募

策略深度:
├─ 兵种搭配 (主将职业+佣兵职业)
├─ 数量 vs 质量选择
├─ 佣兵克制关系利用
└─ 经济资源管理

现代变体:
├─ 简化版: 仅影响属性/外观
├─ 技能版: 佣兵提供额外技能
└─ 召唤版: 战斗中召唤临时单位
```

## 连携与羁绊

### 火纹羁绊系统

```
支援等级:
├─ C级: 基础好感
├─ B级: 中级羁绊
├─ A级: 高级羁绊
└─ S级: 恋人/至亲 (部分作品)

战斗效果:
├─ 相邻时: 属性加成
├─ 概率触发: 协助攻击/防守
├─ 支援对话: 剧情展开
└─ 结局影响: 配对结局

设计意义:
- 强化角色塑造
- 增加战斗策略
- 提升情感投入
- 鼓励多样化组队
```

### 连携攻击设计

```
触发条件:
├─ 位置条件: 特定站位
├─ 时机条件: 特定时机
├─ 状态条件: 特定状态
└─ 组合条件: 特定角色/职业

效果类型:
├─ 伤害加成: 共同攻击增伤
├─ 范围扩大: 攻击影响额外目标
├─ 效果附加: 附加控制/减益
└─ 行动增益: 额外行动/移动

平衡考量:
- 强连携需要高代价 (如特定角色必须上场)
- 避免单一最优解
- 鼓励多样组合探索
```

## 战斗节奏控制

### 回合节奏

```
标准节奏:
玩家回合 → 敌人回合 → 玩家回合...

变体:
├─ 快速回合: 某些单位有额外回合
├─ 插入回合: 特定条件触发额外行动
├─ 连续行动: 连杀/连携延长回合
└─ 回合抢夺: 某些技能可以打断

手游适配:
- 考虑加入自动战斗
- 关键决策点手动操作
- 可跳过已看过多次的动画
```

### 战斗时长控制

```
时长影响因素:

地图大小:
├─ 小地图 (10x10): 5-10分钟
├─ 中地图 (20x20): 10-15分钟
└─ 大地图 (30x30+): 20-30分钟

单位数量:
├─ 少 (3-5人): 快速战斗
├─ 中 (6-8人): 标准体验
└─ 多 (9-12人+): 史诗战斗

敌人配置:
├─ 精英战: 少而精，单场战斗久
├─ 遭遇战: 多而弱，快速清理
└─ 混合战: 节奏变化
```

## 战斗数值设计

### 属性体系

```
一级属性 (基础):
├─ 力量: 影响物理攻击
├─ 技巧: 影响命中/暴击
├─ 速度: 影响闪避/连击
├─ 防御: 影响物理防御
├─ 魔防: 影响魔法防御
└─ 幸运: 影响各种概率

二级属性 (战斗):
├─ 生命值 (HP)
├─ 移动力 (MOV)
├─ 射程 (RNG)
└─ 负重 (携带兵种/装备)

成长率 (火纹模式):
- 每次升级随机提升属性
- 不同职业成长率不同
- 角色有个体差异
```

### 战斗公式参考

```
命中率:
= 技巧 - 敌方速度 + 武器命中 + 地形补正
范围: 5% - 95%

暴击率:
= 技巧/2 + 武器暴击 - 敌方幸运
通常: 3-20%
暴击伤害: 2倍或3倍

伤害计算:
= max(1, 攻击力 - 防御力)
或
= 攻击力 × (1 - 防御力/(防御力+100))

攻速差:
= 我方速度 - 敌方速度
≥5: 可以追击 (攻击两次)
```
