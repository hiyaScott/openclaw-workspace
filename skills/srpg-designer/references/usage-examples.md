# SRPG Designer Skill 使用示例

## 示例1: 设计新英雄技能

### 用户输入
"帮我设计一个SRPG游戏中的火系法师，要求有AOE能力和控制能力"

### Skill响应示例
```
基于621位英雄数据库，为您设计火系法师：

## 定位分析
- 类型: 法师/AOE控制
- 参考角色: 天地劫-夏侯仪、梦幻模拟战-波赞鲁、铃兰之剑-贝拉
- 核心机制: AOE伤害 + 地形控制

## 技能设计

### 天赋【炎之魔导】
- 效果: 对燃烧目标伤害+20%，击杀后制造燃烧地形
- 参考: 夏侯仪【炎之力】、贝拉【晶能亲和】

### 技能1【火球术】
- 倍率: 1.5倍单体
- 效果: 附加燃烧，持续2回合
- 冷却: 2回合
- 参考倍率: 天地劫单体1.5-1.8倍

### 技能2【烈焰风暴】
- 倍率: 0.6倍范围(5格)
- 效果: 范围火伤，制造燃烧地形2回合
- 冷却: 3回合
- 参考倍率: AOE常见0.5-0.7倍

### 技能3【焚天煮海】(气力技)
- 倍率: 0.8倍大范围(7格)
- 效果: 大范围火伤，全场制造燃烧地形3回合
- 消耗: 3点气力
- 参考: 宇文拓【轩辕天罚】、李逍遥【酒神咒】

## 数值建议
| 属性 | 初始 | 成长 | 满级 |
|------|------|------|------|
| 法攻 | 高 | 高 | 顶级 |
| 法防 | 中 | 中 | 中等 |
| 生命 | 低 | 低 | 较低 |
| 移动力 | 3 | - | 3 |
| 射程 | 2-3 | - | 2-3 |

## 阵容搭配
- 推荐队友: 坦克(护卫) + 治疗 + 物理输出
- 克制关系: 克制冰系，被水系克制
- 适用场景: PVE清怪、PVP控场
```

## 示例2: 关卡设计咨询

### 用户输入
"设计一个教学关卡，教玩家如何使用地形机制"

### Skill响应示例
```
基于621位英雄数据库和关卡设计经验，为您设计地形教学关：

## 关卡信息
- 编号: 1-3
- 名称: 高地优势
- 目标: 歼灭所有敌人
- 推荐等级: 3级

## 设计理念
参考铃兰之剑地形系统和天地劫高低差机制：
- 高地攻击低地: +15%伤害
- 低地攻击高地: -10%伤害
- 地形遮挡: 阻挡弹道

## 地图设计
```
[高]    [敌人]
        |
[中] — [平地] — [玩家出生点]
        |
[低]    [敌人]
```

## 敌人配置
| 敌人 | 位置 | 特性 |
|------|------|------|
| 弓箭手x2 | 高地 | 利用高地优势攻击玩家 |
| 步兵x2 | 低地 | 被玩家高地攻击时承受额外伤害 |

## 教学引导
1. 提示玩家"占领高地获得优势"
2. 显示伤害数值对比(高地vs平地)
3. 引导玩家利用地形击败敌人

## 参考案例
- 铃兰之剑: 高低差+15%攻击
- 天地劫: 地形效果多样化
- 皇家骑士团2: 经典高低差系统
```

## 示例3: 数值平衡验证

### 用户输入
"帮我验证这个新设计的刺客是否平衡"

### Skill响应示例
```
基于621位英雄数据库进行数值平衡验证：

## 新刺客数据
- 单体技能: 1.8倍，冷却2回合
- 移动力: 6格
- 机制: 无视护卫，击杀后再动

## 对比分析

| 角色 | 单体倍率 | 移动力 | 再动条件 | 特殊机制 |
|------|----------|--------|----------|----------|
| 新刺客 | 1.8x | 6 | 击杀 | 无视护卫 |
| 曹沁(天地劫) | 1.6-1.8x | 5+突进2 | 击杀 | 闪避+过T |
| 雪莉(梦战) | 1.6x | 5 | 击杀(最多2次) | 飞行 |
| 泽瑞达(梦战) | 1.8x | 5+瞬移 | 击杀 | 无视护卫 |
| 科尔(铃兰) | 1.6x | 6+再动 | 击杀 | 双闪+无视援护 |

## 评估结果

### ✅ 优势
- 倍率与同类角色持平(1.6-1.8倍)
- 再动机制有明确的参考
- 无视护卫是刺客标准配置

### ⚠️ 建议调整
1. 再动次数限制: 参考雪莉"每回合最多2次"
2. 生存能力: 建议增加闪避/隐身机制
3. 克制关系: 明确被弓手/羽士克制

### 📊 机制覆盖率
- 再动: 621位中25.1%有此机制
- 无视护卫: 10.6%有此机制
- 组合后属于高稀有度配置

## 最终建议
- 稀有度建议: SSR/传说
- 生存能力需补充
- 再动次数需限制
```

## 示例4: 机制创新建议

### 用户输入
"有什么创新的战棋机制可以参考？"

### Skill响应示例
```
基于621位英雄数据库的机制分析，提供创新建议：

## 五游戏机制创新对比

| 游戏 | 核心创新 | 机制类型 |
|------|----------|----------|
| 天地劫 | 气力技系统 | 战斗节奏控制 |
| 梦幻模拟战 | 超绝+再动 | 阵营核心机制 |
| 铃兰之剑 | 地形互动 | 环境利用 |
| 三国志战棋版 | 战法联动 | Build深度 |
| 三国望神州 | 合击系统 | 空间协同 |

## 可借鉴的创新机制

### 1. 气力技系统 (天地劫)
- 机制: 强力技能需积攒气力
- 优点: 限制强力技能频率，增加策略深度
- 适用: PVE/PVP平衡

### 2. 超绝系统 (梦幻模拟战)
- 机制: 阵营核心技能，提供攻防增益
- 优点: 强化阵营搭配，增加编队策略
- 适用: 多阵营游戏

### 3. 地形互动 (铃兰之剑)
- 机制: 推/拉/击退改变地形
- 优点: 环境利用，创造无限可能
- 适用: 高低差地图

### 4. 战法联动 (三国志战棋版)
- 机制: 技能组合创造Build
- 优点: 深度策略，高重玩价值
- 适用: 硬核策略游戏

### 5. 合击系统 (三国望神州)
- 机制: 相邻友方同时攻击
- 优点: 空间协同，团队配合
- 适用: 小队战斗

## 机制覆盖率统计
- 再动: 25.1% (高覆盖率，成熟机制)
- 召唤: 12.6% (中等，可创新)
- 地形互动: 10.5% (低覆盖率，创新空间)
- 特殊: 建议开发新机制

## 建议的创新方向
1. 时间系统: 行动顺序可变
2. 天气系统: 影响战场环境
3. 连携系统: 多角色组合技
4. 变形系统: 角色形态切换
```
