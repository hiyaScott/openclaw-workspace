## 🎯 核心机制分析

### 战法联动Build
三国志战棋版的核心深度在于战法组合：
- **自带战法**: 武将固有技能
- **传承战法**: 拆解武将获得
- **事件战法**: 特定事件获得

### 兵种克制链
```
骑兵 → 弓兵 → 枪兵 → 骑兵
盾兵: 独立防御体系
```

### 核心体系
1. **火烧队**: 周瑜+陆逊+黄盖
2. **魏盾队**: 曹操+曹仁+于禁
3. **蜀枪队**: 刘备+张飞+赵云
4. **吴弓队**: 孙权+太史慈+甘宁
5. **群骑队**: 吕布+貂蝉+华佗

## 💡 SLG+SRPG设计启示

1. **稀有度分层陷阱**: 纯数值分层导致低稀有度完全失业
2. **战法独特性**: 存活紫将/边缘橙将往往因独特机制有niche
3. **兵种与势力**: 骑兵>枪兵>盾兵的兵种内卷
4. **战法搭配深度**: 武将强度不仅看个体，更看战法组合
