# 重点产品分析 (Game Analysis)

## 日系SRPG手游

### 《梦幻模拟战》手游

#### 产品概述
- **开发商**: 紫龙游戏
- **上线时间**: 2018年
- **核心卖点**: 经典IP还原 + 手游化创新

#### 核心系统分析

**佣兵系统**:
```
设计亮点:
├─ 主将+佣兵双层单位
├─ 兵种克制关系丰富
├─ 策略搭配深度高
└─ 还原原作特色

手游适配:
├─ 简化操作 (自动战斗)
├─ 佣兵培养长期目标
└─ 不影响战棋核心体验
```

**转职系统**:
```
多分支设计:
├─ 每个角色6+转职方向
├─ 跨系转职保留技能
├─ 终极职业独特定位
└─ 提供高度自由度

策略影响:
- 同一角色不同Build差异巨大
- 鼓励多角色收集培养
- 转职材料成为长期目标
```

**关卡设计**:
```
经典复刻:
├─ 原作5代经典战役
├─ 300+关卡内容
└─ 情怀+新鲜感兼顾

多样目标:
├─ 全歼敌军
├─ 守护友军
├─ 护送撤离
└─ 斩首首领

创新玩法:
├─ 无尽航路 (Roguelike)
├─ 世界BOSS
├─ 竞技场
└─ 公会战
```

**养成系统**:
```
深度养成:
├─ 角色等级/星级
├─ 职业进阶
├─ 装备强化/精炼
├─ 兵种训练
├─ 羁绊系统
└─ 附魔系统

付费设计:
├─ 英雄抽卡 (核心付费)
├─ 装备抽卡
├─ 皮肤付费
└─ 月卡/基金
```

#### 成功要素
1. **IP还原**: 漆原智志人设、经典关卡、BGM
2. **策略深度**: 兵种克制、地形利用、技能搭配
3. **长线养成**: 多维度养成，持续追求
4. **内容充足**: 300+关卡，持续更新

### 《天地劫：幽城再临》

#### 产品概述
- **开发商**: 紫龙游戏
- **上线时间**: 2021年
- **核心卖点**: 天地劫IP + 电影级剧情

#### 核心系统分析

**五魂化蕴系统**:
```
原作还原:
├─ 五魂代表五种属性
├─ 化蕴解锁技能
├─ 属性之间相互关联

手游简化:
├─ 增加法防属性
├─ 双向克制改单向
├─ 简化计算但不失深度
```

**关卡设计**:
```
叙事融合:
├─ 所有关卡都有剧情
├─ 小型关卡不敷衍
├─ 电影化演出
└─ 手游中罕见的高投入

环境互动:
├─ 移动力蘑菇
├─ Debuff地形
├─ 高低差利用
└─ 机关解谜元素
```

**归真模式**:
```
硬核挑战:
├─ 固定角色和数值
├─ 不受卡牌养成影响
├─ 纯策略考验
└─ 致敬原作难度

意义:
- 满足硬核玩家
- 展示策略深度
- 与养成关卡形成对比
```

#### 与《梦幻模拟战》差异

```
梦战 → 天地劫

画风:
├─ 梦战: 日式奇幻，漆原智志风格
└─ 天地劫: 国风武侠，水墨风格

剧情:
├─ 梦战: 多线剧情，关卡承载有限
└─ 天地劫: 主线剧情，每关都有叙事

系统:
├─ 梦战: 佣兵系统
└─ 天地劫: 五魂化蕴

养成:
├─ 梦战: 深度且复杂
└─ 天地劫: 相对简化
```

### 《铃兰之剑》

#### 产品概述
- **开发商**: 心动网络
- **特点**: 像素风 + 单机+网游混合

#### 核心亮点

**美术风格**:
```
像素HD:
├─ 复古像素风格
├─ HD-2D光影效果
├─ 精美角色立绘
└─ 独特视觉记忆点
```

**玩法模式**:
```
单机模式:
├─ 完整单机剧情
├─ 一次性付费体验
├─ 无养成压力
└─ 强调策略深度

网游模式:
├─ 角色收集养成
├─ 持续更新内容
├─ 社交竞技玩法
└─ 长线运营设计

融合设计:
- 两种模式独立但互通
- 满足不同玩家需求
- 创新商业模式
```

## 经典参考

### 《火焰纹章》系列

#### 核心机制

**武器三角**:
```
剑 > 斧 > 枪 > 剑

设计意义:
- 简单易懂
- 策略基础
- 增加职业价值
```

**Permadeath (永久死亡)**:
```
硬核设定:
- 角色死亡不可复活
- 高风险高投入
- 每个决策都重要

现代妥协:
-  casual模式可关闭
- 时间回溯功能
- 降低门槛但保留硬核选项
```

**角色羁绊**:
```
支援系统:
├─ 相邻作战积累好感
├─ 支援对话剧情
├─ 战斗属性加成
└─ 影响结局配对
```
