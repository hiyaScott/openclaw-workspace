# 网络游戏设计师 Skill

## 创建时间
2026-03-12

## 能力范围
专业级网络游戏设计能力，涵盖：

1. **游戏设计文档 (GDD)** - 完整设计文档编写
2. **系统架构设计** - MMORPG服务器架构、分区分服
3. **经济系统设计** - 货币体系、产出消耗、通胀控制
4. **社交系统设计** - 公会、好友、组队、PVP/PVE
5. **数值策划** - 战斗数值、经济数值、成长曲线
