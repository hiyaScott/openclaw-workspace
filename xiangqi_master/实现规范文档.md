## 四、多步关卡规范

### 4.1 数据结构

```javascript
{
  multiStep: true,
  steps: [
    {
      // 单目标
      target: { x: 6, y: 6 },
      msg: '第一步完成！'
    },
    {
      // 多目标（多个正确答案）
      target: [{ x: 6, y: 6 }, { x: 3, y: 5 }],
      msg: '很好！继续！'
    },
    {
      // 吃子目标
      target: { x: 5, y: 4, type: 'capture' },
      msg: '完美！'
    }
  ]
}
```

### 4.2 判定逻辑

```javascript
function checkStepComplete(target, currentStep) {
  if (Array.isArray(currentStep.target)) {
    // 多目标匹配
    return currentStep.target.some(t => 
      target.x === t.x && target.y === t.y
    );
  } else {
    // 单目标匹配
    return target.x === currentStep.target.x && 
           target.y === currentStep.target.y;
  }
}
```
