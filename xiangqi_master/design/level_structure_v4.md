
# 象棋大师 - 关卡结构 v4（修正版）

## 四层结构定义

```
【章】→【主题】→【关】→【局】
  │       │       │      │
 10章    n主题   9关    8-12局
         /章     /主题  /关
```

**关键修正：**
- 【关】= 教学内容（如"移动车"、"移动炮"）
- 【局】= 连续的8-12局练习（之前的"关卡"概念）
- 完成一关后显示【经验】【胜率】【时间】统计

## 游戏数据估算

```
总章数: 10章
总主题: ~120个
总关卡: 120主题 × 9关 = 1080关
总局数: 假设平均每关10局
       1080关 × 10局 = 10800局

预估游戏时长: 每局1-3分钟
              10800局 × 2分钟 = 360小时
              约等于 6个月 每日2小时的学习量
```
