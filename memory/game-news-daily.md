# 🎮 游戏资讯日报 - 2026年3月4日

## 📰 头条新闻

### 1. 米哈游UE5 3D MOBA预研项目曝光
米哈游官方招聘网站更新一批预研项目岗位，**UE5 3D MOBA新项目**正式浮出水面。本次招聘覆盖交互策划负责人、UE5客户端开发、游戏服务器开发、测试开发等核心职位，项目采用虚幻引擎5研发，定位3D风格化写实美术路线，主打第三人称动作战斗。

### 2. 游卡网络《元夜十二谈》首曝
《三国杀》研发商游卡网络正式官宣自研新作——**新志怪民俗题材策略游戏《元夜十二谈》**，以"十二轮转，月碎傩生"为核心主题，构建融合现代都市与中式志怪的"天枢城"世界观。玩家将扮演拥有"重瞳"的风水侦探，调查悬剑桥、锁龙井、兽面拍卖场等民俗怪谈事件。

### 3. 开放世界新游春季之战
