   - 设置窗口尺寸 1280x720

   - WASD移动
   - 空格冲刺（3秒冷却）
   - 生命值系统

5. **敌人系统**
   - 基础敌人AI (`enemy.gd`)
   - 敌人生成管理器 (`enemy_manager.gd`)
   - 敌人在领地内持续掉血
   - 敌人尝试逃离领地

6. **UI界面** (`ui.gd`)
   - 生命值条
   - 分数显示
   - 占领率显示
