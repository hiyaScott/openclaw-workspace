- `examples/deepspace-theme-comparison.html` - 左右对比展示

### 主题切换机制
```css
/* 默认 DeepSpace (深色) */
:root { --bg-primary: #0a0a0f; ... }

/* Daylight (明亮) - 添加 .daylight 类到 body */
body.daylight { --bg-primary: #f5f5f7; ... }
```

### JavaScript 切换
```javascript
function toggleTheme() {
  const isDaylight = document.body.classList.toggle('daylight');
  localStorage.setItem('theme', isDaylight ? 'daylight' : 'dark');
}
```
