# 妈妈计数器硬件方案升级 - v1.1.0 开箱即用版 (2026-03-21)

## 升级背景
Scott要求"开箱即用"效果，因此将原方案的 ESP32-C3-Display + 外置MAX9814麦克风 升级为 一体化方案。

## 新旧方案对比

| 项目 | v1.0.0 旧方案 | v1.1.0 新方案 |
|------|---------------|---------------|
| 开发板 | ESP32-C3-Display (~￥55) | 微雪 ESP32-S3 1.85寸 (~￥80-100) |
| 屏幕 | 1.28寸 240×240 | 1.85寸 360×360 (更大更清晰) |
| 麦克风 | MAX9814 外置 (需焊接) | 板载I2S数字麦 (即插即用) |
| 陀螺仪 | 无 | QMI8658 (支持翻转静音) |
| 外壳 | 需3D打印/DIY | CNC铝合金 (开箱即用) |
| 总价 | ~￥80 | ~￥100 (+￥20) |
| 工作量 | 高 | 低 |

## 核心改进

### 1. 硬件一体化
- 屏幕+麦克风+陀螺仪+外壳，全部集成在一块开发板上
- 无需焊接，无需3D打印外壳
- CNC铝合金外壳，质感更好

### 2. 新增功能
- **翻转静音**: 屏幕朝下自动暂停监听
- **更大屏幕**: 1.85寸360p，数字显示更清晰
- **数字麦克风**: I2S接口，音质比模拟麦克风更好

### 3. 开发简化
- 引脚全部预定义，无需查手册
- 提供完整的Arduino代码(I2S麦克风驱动)
- 开箱后0.5天即可验证硬件
