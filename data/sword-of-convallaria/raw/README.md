## 数据字段说明

每位英雄包含以下完整数据：

```json
{
  "id": "英雄唯一标识",
  "name_cn": "中文名",
  "name_en": "英文名",
  "rarity": "稀有度(星级)",
  "movement": "移动力",
  "jump": "跳跃力",
  "weapon_type": "武器类型",
  "special_job": "特职",
  "personality": "个性技能",
  "stats": {
    "level_1": { "1级基础属性" },
    "max_level": { "满级属性" }
  },
  "nickname": "玩家昵称",
  "cv_cn": "中文配音",
  "cv_jp": "日文配音",
  "description": "角色背景"
}
```

### 属性字段
- **hp**: 生命值
- **phys_atk**: 物理攻击
- **phys_def**: 物理防御
- **mag_atk**: 魔法攻击
- **mag_def**: 魔法防御
- **speed**: 速度
